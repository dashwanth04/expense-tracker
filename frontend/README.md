# Expense Tracker (Angular 18)

Starter Angular 18 expense tracker app.

## Features
- Add expenses (description, amount, category, date)
- List expenses and delete entries
- Filter by month
- Monthly totals summary
- Data persisted in localStorage
- Bootstrap 5 responsive dark/light theme

## Setup
1. Extract the ZIP.
2. Open the project folder in VS Code.
3. Run:
   ```
   npm install
   ng serve
   ```
4. Open http://localhost:4200

Notes:
- This template uses localStorage for data persistence. For multi-device sync, add a backend API.
